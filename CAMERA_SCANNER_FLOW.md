## üîÑ Complete Request-Response Flow

### Request Flow (Client ‚Üí Server ‚Üí ML)
```
1. User Action: Click üì∏ Capture or üü¢ Start Detection
   ‚îî‚îÄ> Screenshot from video or image from upload

2. Frontend (Vue.js)
   ‚îî‚îÄ> Convert image to Base64
   ‚îî‚îÄ> Send POST to /api/staff/detect-cocolumber
   ‚îî‚îÄ> Include JWT Token in Authorization header
   ‚îî‚îÄ> Send JSON: { image: "data:image/png;base64,..." }

3. Backend (Node.js)
   ‚îî‚îÄ> Verify JWT token (is user logged in?)
   ‚îî‚îÄ> Check user role (is staff or admin?)
   ‚îî‚îÄ> Receive Base64 image
   ‚îî‚îÄ> Forward to ML service: POST http://localhost:5000/predict
   ‚îî‚îÄ> Wait for response (max 30 seconds)

4. ML Service (Python TensorFlow)
   ‚îî‚îÄ> Decode Base64 image
   ‚îî‚îÄ> Load TensorFlow MobileNetV2 model (first time: ~3-5 sec)
   ‚îî‚îÄ> Preprocess image (224x224, normalize)
   ‚îî‚îÄ> Run inference (top 10 predictions)
   ‚îî‚îÄ> Classify: Is it human? wood? other?
   ‚îî‚îÄ> If wood: estimate measurements
   ‚îî‚îÄ> Return JSON result

5. Backend (receives ML response)
   ‚îî‚îÄ> Validate response
   ‚îî‚îÄ> Log results to console
   ‚îî‚îÄ> Send response back to frontend

6. Frontend (receives results)
   ‚îî‚îÄ> Update scanResults data
   ‚îî‚îÄ> Re-render Vue component
   ‚îî‚îÄ> Display results to user
```
